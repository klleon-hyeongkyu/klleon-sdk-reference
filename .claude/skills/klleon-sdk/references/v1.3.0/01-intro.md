# Klleon Chat SDK 소개

## 개요

Klleon Chat SDK는 웹 애플리케이션에 **실시간 인터랙티브 아바타를 손쉽게 통합할 수 있도록 제작된 JavaScript 라이브러리**입니다.

## 주요 특징

- **간편한 연동**: 스크립트 태그 추가로 빠른 설치 및 직관적 API 제공
- **실시간 양방향 통신**: 사용자와 아바타 간 텍스트 및 음성 메시지 실시간 교환
- **커스터마이징 가능한 UI**: 기본 제공 컴포넌트 또는 사용자 정의 UI 구성 지원
- **다양한 SDK 이벤트 구독**: 상태 변경 및 채팅 이벤트 활용 가능
- **TypeScript 타입 정의 제공**: `.d.ts` 타입 정의 파일을 제공하여 타입 안정성 향상

## 대상 사용자

- 웹 개발자
- 프로덕트 매니저/기획자
- SDK 활용을 궁금해하는 모든 사용자

## 기능 구성

SDK는 LLM, TTS, 립싱크 기능을 포함하여 완전한 아바타 채팅 경험을 제공합니다.

> **참고**: 전체 문서는 설치부터 고급 활용법까지 다루고 있습니다.
