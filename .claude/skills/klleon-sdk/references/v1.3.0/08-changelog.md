## 버전 히스토리

### v1.3.0

최신 안정 버전입니다.

**새로운 기능:**
- `wakeUpAvatar()` 메서드 추가 - 대기 상태 아바타 활성화
- `onErrorEvent()` 이벤트 리스너 추가 - 에러 처리 콜백 지원

**새로운 이벤트 타입:**
- `RATE_LIMIT` - 메시지 수신 비활성화 (요청 제한)
- `ACTIVATE_VOICE` - 음성 인식 활성화
- `HANDOVER_START` - 핸드오버 시작
- `HANDOVER_SUCCESS` - 핸드오버 성공
- `HANDOVER_FAIL` - 핸드오버 실패

**새로운 상태:**
- `DESTROYED` - SDK 완전 종료 상태
